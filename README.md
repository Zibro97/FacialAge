# FacialAge
> 주어진 얼굴 사진을 통해 실제 나이를 예측하는 모델. 학교 딥러닝 프로젝트
- 개발 기간 : 2020.11.29~2020.12.14
- 사용 언어 : python
- 라이브러리 : `pytorch`, `torchvision`, `opencv`, `matplotlib`, `numpy`
- 학습 알고리즘 : EfficientNet
- 학습 정확도 : 61%
